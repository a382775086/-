# vue2


基于VUE2的一个单页面音乐播放网页
===


实现功能：
-------
1.暂停、播放、下一首、上一首

2.进度条的拖动

3.播放列表的打开、关闭以及歌曲添加至播放列表

4.播放列表歌曲的切换

5.歌手个人页面

6.歌词显示与展开

（仅收录了每个歌手中的前两首）


项目运行：
-------
* npm i
* npm run dev
